# Doa Tech 🤝💻

## 📌 Descrição
O Doa Tech é uma iniciativa criada com o objetivo de amparar indivíduos em momentos delicados, conectando pessoas que precisam de ajuda com aquelas que desejam contribuir.

A proposta do projeto é utilizar a tecnologia como ponte entre doadores e beneficiários, tornando essa conexão simples, rápida e acessível para todos.

## 🎯 Objetivo
Facilitar o processo de doação, promovendo solidariedade por meio de uma plataforma digital intuitiva e eficiente.

## ⚙️ Funcionalidades
- Cadastro de usuários
- Cadastro de pedidos de ajuda
- Cadastro de doações
- Listagem de pedidos disponíveis
- Conexão entre doadores e solicitantes
- Validação de informações

## 🛠️ Tecnologias utilizadas
- PHP
- MySQL
- HTML/CSS

## 🚀 Como executar
1. Clonar o repositório
2. Configurar o banco de dados
3. Executar em servidor local (ex: XAMPP)
